    • About HTML

    HTML stands for Hyper Text Markup Language it is a language for specifying how text and graphics appear on a web page, in simple word it is the standard markup language for creating Web pages, HTML describes the structure of a Web page, web browser retrieves the HTML web page and renders it accordingly the first version of HTML was written by Tim Berners-Lee in 1993.

    HTML versions and released date

    Version Id Release Date
    HTML 1993
    HTML 2 1995
    HTML 3 1997
    HTML 4 2000
    HTML 5 2014

    Sample HTML Code

           <!DOCTYPE html>
              <html>
                     <head>
                            <title>Website title goes here</title>
                     </head>
                     <body>
                            //your code goes here
                     </body>
              </html>

    Some Basic HTML tags & their use

    Tag name How to Implement Explanation
    h1, h2, h3... h6 tags <h1> </h1> The h1 to h6 tags are used to define HTML headings.
    p tag <p> </p> The p tag defines a paragraph.
    a tag <a> </a> The a tag defines a hyperlink, which is used to link from one page to another.
    div tag <div> </div> The div tag defines a division or a section in an HTML document.

    HTML এর সম্বন্ধে

    HTML এর পুরো কথাটি হল Hyper Text Markup Language এটি ওয়েবপেজে টেস্কট এবং গ্রাফিক্স কীভাবে দেখাবে তা নির্দিষ্ট করার জন্য একটি ভাষা, এক কথায় ওয়েবপেজ তৈরি করার জন্য চিহ্নিত করার ভাষা, HTML একটি ওয়েব পেজের গঠন বর্ণনা করে, ব্রাউজার HTML নিয়মাবলীকে সাজিয়ে এবং সেই অনুযায়ী ওয়েব পেজকে রেন্ডার করে

    কিছু সাধারণ HTML ট্যাগ এবং তাদের ব্যবহার

    ট্যাগের নাম কিভাবে বাস্তবায়ন করা হয় ব্যাখ্যা
    h1, h2, h3... h6 tags <h1> </h1> h1 থেকে h6 ট্যাগগুলি ব্যাবহিত হয় শিরোনাম লেখার ক্ষেত্রে।
    p tag <p> </p> অনুচ্ছেদ লেখার ক্ষেত্রে p ট্যাগ ব্যাবহিত হয়।
    a tag <a> </a> a বা অঙ্কার ট্যাগ একটি পেজের সাথে আর একটি পেজকে সংযুক্ত করতে ব্যাবহিত হয়।
    div tag <div> </div> div ট্যাগ html ডকুমেন্টকে একটি বিভাগে পরিণত করতে ব্যাবহিত হয়।
